# 🎉 Abdulloh's Birthday Surprise Web Project
This is special vibeb coded simple birthday repo for my friend.
A creative, interactive web-based birthday greeting specially designed for my friend **Abdulloh**. This project combines web development basics with a touch of gaming culture (Minecraft) to create a memorable surprise.

## 🚀 How it Works

1. **The Entry:** The user is greeted with a clean landing page asking to open a gift.
2. **The Countdown:** Upon clicking, a 20-second countdown begins with:
   - Synchronized background music.
   - A photo carousel showcasing memories (rotating every 3 seconds).
   - Dynamic birthday messages in various colors (changing every 5 seconds).
3. **The Creeper Event:** At 0 seconds, the music stops, a **Creeper** appears, and a dramatic explosion effect occurs.
4. **The Reveal:** After the explosion, a final "HAPPY BIRTHDAY Abdulloh" greeting is displayed in a stylish layout.

## 🛠️ Built With

* **HTML5:** Structural layout.
* **CSS3:** Styling, animations, and image carousel.
* **JavaScript:** Timer logic, music synchronization, and the "Creeper" chroma-key effect (Canvas API).
